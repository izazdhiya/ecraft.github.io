# ecraft.github.io

Pemasangan aplikasi :
- Download semua file
- Rename folder yang didownload menjadi 'ecraft'
- Tambah database baru di phpmyadmin dengan nama 'ecraft' kemudian masukkan file ecraft.sql
- Nyalakan XAMPP
- Buka aplikasi melalui browser
