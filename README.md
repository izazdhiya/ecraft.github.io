# E Craft 

Pemasangan aplikasi :
- Download semua file
- Ekstrak ke direktori htdocs pada XAPP
- Rename folder yang diekstrak menjadi 'ecraft'
- Tambah database baru di phpmyadmin dengan nama 'ecraft' kemudian masukkan file ecraft.sql
- Nyalakan XAMPP
- Buka aplikasi melalui browser
